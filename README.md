# Blog Post Project

This project has been restructured into two separate applications:

- **admin-app**: For managing blog posts (creating, editing, deleting).
- **user-app**: For viewing blog posts.

Both applications are developed using Bootstrap for responsive design.

## Admin App
Located in `admin-app/` folder. Provides a form to add new posts and list existing ones.

## User App
Located in `user-app/` folder. Displays blog posts in a card layout.

To run the apps, open the respective `index.html` files in a web browser.
