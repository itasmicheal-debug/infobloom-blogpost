# Blog Post Project

A unified blog post application with separate user and admin interfaces, built with Bootstrap for responsive design.

## Features
- **User Interface**: View blog posts in a clean card layout
- **Admin Interface**: Create, edit, and delete blog posts
- **Shared Data**: Posts are stored in browser localStorage
- **Navigation**: Easy switching between user and admin views

## Getting Started
1. Open `user-app/index.html` in your web browser to view posts
2. Use the "Admin" link in the navigation to manage posts
3. Posts created in admin will appear in the user view

## Apps
- **user-app**: For reading blog posts
- **admin-app**: For managing blog posts

Both apps share the same data via localStorage.
